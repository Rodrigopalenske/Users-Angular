# Users-Angular
Projeto para cadastro de usuarios

---
## FrontEnd

## Funcionalidades
- Tela de formulário cadastro do usuário

## Tecnologias Utilizadas

- **TypeScript**
- **Angula** - Framework frontend para criar uma interface
- **Angular Material** -  Biblioteca de componentes para interface no Angular


---
## BackEnd

## Funcionalidades
 - Criação de usuários com validações e inserção no banco de dados

## Tecnologias Utilizadas

- **Java 21**
- **Spring Boot** — Framework web para criação de rotas
- **Spring Data JPA** — Acesso e manipulação de dados
- **Spring Web** — Criação de APIs REST
- **Lombok** — Auxilio no desenvolvimento da API
- **H2** — Banco de dados relacional para ambiente de desenvolvimento
- **bcrypt** — Criptografia de senhas
